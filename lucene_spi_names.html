SPI Names in Lucene 8.3.0

The SPI names can be used to build custom analyzers, for example: 
Analyzer analyzer = CustomAnalyzer.builder()
        .withTokenizer("icu")
        .addTokenFilter("lowercase")
        .addTokenFilter("asciiFolding")
        .build();
